fromData:Object  [read-only]

The data that's extracted from the current location. First time a route is changed this will be null.

Implementation
    public function get fromData():Object

Example
If at route "/page/{pageName}/{pageNumber}/" and the current location is "/page/hello/12/" the fromData will be { pageName: "hello", pageNumber: 12 }

toDataproperty 
toData:Object  [read-only]

The data that's extracted from the goto call. This is the same as in fromData except that this also contains any arbitrary data passed along with the RouteMachine.GOTO call.

Implementation
    public function get toData():Object

fromData

Example
So if we call RouteMachine.GOTO with the body with something like { pageName: "hello", pageNumber: 12, notInRoute: "other stuff" } the notInRoute variable will only exist in the toData, not in the fromData.
